Introduction¶

Stock prices are commonly affected by multiple different macroeconomic factors. This can range from things like unemployment dipping and peaking to the steady climb of the United States' GDP. Thus, predicting the outcome of stocks can be elusive and highly valuable endeavor.

This project in particular aims to leverage historical financial data alongside macroeconomic data in order to potentially create a model for stock price prediction. By integrating these two types of datasets, we seek to understand whether and how macroeconomic trends can affect stock prices over time.

Project Goals¶

The goal of this project is to analyze the correlation between the stock market and macroeconomic factors. With this information we plan to create a machine learning model that will be able to predict stock prices of companies based on external market factors.

Datasets¶

The first dataset we plan on using is yfinance, which is a python library that features data from Yahoo Finance. This library features a variety of datasets, mainly being stock market data. Using this library, we can extract past data from the S&P 500 or from individual companies. Additionally we plan to use multiple datasets from the FRED. FRED offers historic economic data, such as past unemployment rates, inflation rates, and GDP history.

We are hoping that by incorporating both of these datasets we will be able to examine the effects certain economic events have on the stock market. By creating a model that can analyze the connections between both of these datasets, we will be able to use past trends of an individual company, and predict whether their stock will go up or down, based on predictions in the economy. We can also use this in the bigger picture, predicting whether the stock market will go into an expansion or recession. For example, being able to predict how much the stock market will drop based on a prediction of a 2% increase in unemployment. We can further use this to even analyze correlation between specific industries, and whether certain industries are more sensitive to things like inflation increases or other factors.

S&P 500 Data¶

The following code uses the yfinance libary to extract data relating to the S&P 500. The S&P 500 includes the 500 largest companies on the stock market. Because these companies are from various different industries, it provides a good indicator on how the overall economy is performing. Below, we plotted the closing prices of the S&P 500 from when they started in 1957 to present day. By looking at this graph we can get an idea of the economy over the years, with an example being the drop in the 2008 economic crisis or the sharp decrease in 2020 due to COVID.

One interesting statistic from this is that 50th percentile is actually at a close price of ~360.16, highlighting just how exponential the stock price has grown within the past ~25 years.

import yfinance as yf

start_date = "1957-01-01"
end_date = "2025-01-01"

# S&P 500 data (GSPC is the ticker (abbreviation) Yahoo Finance uses for the S&P 500)
sp500_data = yf.download("^GSPC", start=start_date, end=end_date)

# Checks if a column consists of a tuple ex: ('Close', 'GSPC')
# and combines it with Close_GSPC
def flatten_col(col):
    return col[0] if col[1] == "" else f"{col[0]}_{col[1]}"

sp500_data.columns = [flatten_col(col) for col in sp500_data.columns.values]

# Reset index to make the date a seperate column instead of the index
sp500_data.reset_index(inplace=True)

# Renames the columns appropriately
tidy_sp500 = sp500_data[['Date', 'Close_^GSPC']].rename(columns={'Close_^GSPC': 'SP500_Close_Price'})

tidy_sp500df = pd.DataFrame(tidy_sp500)
# Making the dataset into a DataFrame to see some general summary statistics
print(tidy_sp500df.describe())
print('\n')


# Grabbing the head and the tail
print(tidy_sp500.head())
print('....')
print(tidy_sp500.tail())

print('\n')

# Create plot of S&P 500 closing prices
plt.plot(tidy_sp500['Date'], tidy_sp500['SP500_Close_Price'], color='blue', linewidth=2)
plt.xlabel('Date by Year', fontsize=14)
plt.ylabel('S&P 500 Close Price', fontsize=14)
plt.title('S&P 500 Close Prices (1957 - 2025)', fontsize=16)
plt.grid(True)
plt.show()

FRED Data¶

FRED offers United States economic data. This will be used to compare with individual stocks and overall stock market in the future. Currently we have taken in data about the CPI, which is consumer price index and pretty much follows inflation rates. We have also extracted the GDP, which is gross domestic product, and the unemployment rates. By using this data we can find interesting statistics such as how GDP rates have been able to go from 934 to 30000 throughout time. Another interesting statistic is grouping unemployment by decades to see that the 1980's was the period with the highest unemployment rate. We can also find general relationships such as CPI and GDP % tending to rise and drop with each other, showing that inflation comes when the economy is performing well.

#Import CSV files

unemployment = pd.read_csv('schaferkotter-freed.github.io/UNRATE.csv')
gdp = pd.read_csv("https://raw.githubusercontent.com/schaferkotter-freed/schaferkotter-freed.github.io/main/GDP.csv")
cpi = pd.read_csv("https://raw.githubusercontent.com/schaferkotter-freed/schaferkotter-freed.github.io/main/CORESTICKM159SFRBATL.csv")


#Tidy variable names

#CPI
cpi = cpi.rename(columns={
    'observation_date': 'Date',
    'CORESTICKM159SFRBATL': 'CPI'
})
#Make pandas know it is a column of dates to make it more interpretable
cpi['Date'] = pd.to_datetime(cpi['Date'])

# GDP
gdp = gdp.rename(columns={
    'observation_date': 'Date',
    'GDP': 'GDP'
})
gdp['Date'] = pd.to_datetime(gdp['Date'])

#Unemployment
unemployment = unemployment.rename(columns={
    'observation_date': 'Date',
    'UNRATE': 'Unemployment_Rate'
})
unemployment['Date'] = pd.to_datetime(unemployment['Date'])


#Merges CPI, GDP, and unemployment rate into 1
merged_df = pd.merge(unemployment, gdp, on='Date', how='inner')
merged_df = pd.merge(merged_df, cpi, on='Date', how='inner')
merged_df.sort_values('Date', inplace=True)

#Find percentage changes between GDP, unemployment rate, and CPI
merged_df['Unemployment_Rate_%'] = merged_df['Unemployment_Rate'].pct_change() * 100
merged_df['GDP_%'] = merged_df['GDP'].pct_change() * 100
merged_df['CPI_%'] = merged_df['CPI'].pct_change() * 100
merged_df = merged_df.dropna()
display(merged_df)
print('\n')

#Stats on GDP
GDP_stats = merged_df['GDP'].describe()
print(GDP_stats)
print('\n')


# Average unemployment rate by decade
merged_df['Year'] = merged_df['Date'].dt.year
merged_df['Decade'] = (merged_df['Year'] // 10) * 10
avg_unemployment_by_decade = merged_df.groupby('Decade')['Unemployment_Rate'].mean()
print(avg_unemployment_by_decade)
